What This Chart Shows

Monthly import values from the top 20 US trading partners. Each line represents a country; color is assigned based on total trade ranking. Dotted vertical lines mark tariff events. The legend shows all countries with the ability to isolate specific ones.

How to Use It

  • Click legend: Click a country name to toggle its visibility
  • Double-click legend: Isolate a single country (double-click again to reset)
  • Hover: See all country values at any date with unified tooltip
  • Compare: Look for divergent patterns after tariff events
